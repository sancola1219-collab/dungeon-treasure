# 地牢尋寶：沉默王陵

第一人稱 3D 地牢探索尋寶遊戲（繁體中文）。無名古王的墓穴「沉默王陵」，
殉葬品堆積如山，但「王的東西」不能碰——尋寶人江無涯為了還債闖入這座傳說裡沒人走出來的陵墓。

**線上遊玩：https://sancola1219-collab.github.io/dungeon-treasure/**

## 玩法

- **兩章戰役**：上層墓道（收集寶藏、找青銅鑰匙）→ 王陵心臟（王棺大廳 Boss「不朽王」、陵墓塌陷倒數逃生）
- **尋寶系統**：金幣袋 / 血紅寶石 / 黃金神像 / 沉默王冠——收集率與價值計入通關評價（S–D）
- 六種武器、八種墓穴守衛（殭屍/古墓犬/獵手/天花板守墓人/守財藤/巨蛛/脹屍/不朽王）
- 火把光照、石磚地牢氛圍、打字機存檔、小地圖、三種難度、手機觸控

## 操作

WASD 移動．滑鼠視角．左鍵攻擊．E 互動/拾取/讀文件．R 裝填．數字鍵/Q 切武器．Tab 物品欄．M 地圖．Esc 暫停
手機：左搖桿移動（推滿奔跑）．滑動畫面轉視角．右下按鈕操作

## 開發

```
npx http-server -p 8126 .   # 本地執行
node --test                 # 跑測試
```

接手開發請讀 `CLAUDE.md`（＝`AGENTS.md`）的設計與審核規範。
